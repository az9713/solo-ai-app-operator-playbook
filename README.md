# Solo AI App Operator Playbook

A practical operating system for solo founders building AI-powered products.

## Inspiration

This playbook was inspired by the YouTube video:

**[8 Vibe Coded Apps That Are Making $1M+/Month](https://www.youtube.com/watch?v=zNOunnM1jTs&t=15s)**

The video covers eight startups built by non-developers using AI tools — Medv, Cal AI, Wave AI, Fly Peter, TrendFeed, Aura, Sleek, and Siteshore/CiteSure — and examines how each scaled to significant revenue.

## About the Playbook

The playbook was created by **ChatGPT 5.4 Thinking** by distilling the lessons learned from the eight startups featured in the video. It compresses recurring patterns across those case studies into a structured, actionable framework for solo AI app builders.

## What's Inside

- **Core doctrine** — judgment over coding, composition over greenfield builds
- **12-step execution loop** — from idea to launch
- **Default tech stack** — Next.js, Supabase, Vercel, Stripe, and more
- **Prompting workflow** — four prompt classes, model routing, prompt hygiene rules
- **Distribution system** — channel-product fit, launch sequence, non-paid growth
- **Reliability gates** — when to harden by stage
- **Failure modes** — seven ways solo AI founders blow up
- **30-day launch plan** — day-by-day execution
- **Idea scoring rubric** — 10-axis scoring (42+ = strong candidate)
